## 🚀 Key Features

- ✅ **Responsive layout** for all screen sizes
- 🎨 **Color variant** options
- ⭐ **Star rating** section using Remix Icons
- 🛒 **Add to cart** call-to-action
- 🧥 **Clean, minimal design**
- 📱 **Mobile-first CSS media queries**
